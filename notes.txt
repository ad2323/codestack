  Sa mga di pa nakainstall last meeting install na kayo ng vue and quasar cli
  Vue cli = npm install -g @vue/cli (for npm only)
  yarn global add @vue/cli
  (for yarn only)
  Quasar cli = npm install -g @quasar/cli (npm)
  yarn global add @quasar/cli (yarn)

  To check meron na si vue and quasar
  vue -V
  quasar -v
  Pag may number after tinype niyo sa terminal, it means na install niyo na

  To create a app type "quasar create <yourprojectname>"
  Sa setup niyan no capital letters and spaces sa project name instead use - for two or more words
  Yung mga description pwede niyo na i enter to proceed
  For selecting features select only Vuex and Axios. Arrow keys to navigate, space to select and deselect, enter to proceed.

  If they ase to install using npm or yarn, kung nag npm install, use npm if yarn add use yarn. Patience till pag na install.

  After may makikita kayo na to get started. Pwede niyo na i open yung project niyo sa visual studio code, then open terminal then type quasar dev.
  Pag updated browser niyo minsan nagshoshow na agad. Pag hindi makikita niyo yung link sa terminal.
